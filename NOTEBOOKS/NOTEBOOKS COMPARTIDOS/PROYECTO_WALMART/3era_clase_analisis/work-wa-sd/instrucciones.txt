Escenario: 
Has realizado una petición de datos historicos a un cliente nuevo. 

Estas esperando un lote de datos que consiste en:
1. Un catalogo de productos
2. Un catalogo de tiendas
3. Un archivo que registra las ventas entre el proveedor (ejemplo WALMART) al consumidor final (la persona comprando en el supermercado).

Basicamente queremos ver qué producto se vendío que en qué día en qué tienda a qué precio y cuantos unidades se vendío. 
Con este información podremos ver el comportamiento de los productos de nuestro cliente nuevo durante el tiempo.

Estos datos deben tener ventas para todos los productos del catálogo de productos.
Estos datos deben cubrir las ventas en todo el país de dos grupos de supermercados.
Estos datos deben tener ventas diarias por producto.

Tu tarea es:
Dentro de un notebook de jupyterhub: 
1. Recopilar los datos que este cliente nuevo nos ha entregado. 
2. Realizar una primera limpieza si hace falta.  
3. Realizar comprobaciones sobre el contenido de los archivos y su integridad entre ellos.
4. Presentarnos lo que encuentres. Tenemos lo que pedimos? Nos falta algo?

Tips:
1. Se puede descargar todos los archivos desde aqui, pinchando code y descargar .zip. 
2. Queremos ver tu estilo de código, cómo piensas y cómo explicas por qué has hecho lo que has hecho.
3. Si a los datos les falta algo, o algo está mal dentro de los datos, cómo los comunicarías al cliente lo más sencillo posible.
